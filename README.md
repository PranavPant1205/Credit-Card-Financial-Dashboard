💳 Credit Card Financial Dashboard
🚀 Overview
Developed an end-to-end Credit Card Analytics Dashboard using Power BI, SQL, and Excel to analyze customer behavior, transaction trends, and revenue performance.
Built a complete data pipeline by creating and managing the database in SQL and loading structured data into Power BI for visualization and insights.

🛠️ Tech Stack
Power BI – Data visualization & dashboard development
SQL – Database creation, data loading, and querying
Excel – Data preprocessing and validation

⚙️ Data Engineering (SQL Work)
Designed and created database schema for credit card data
Imported and loaded datasets into SQL tables
Performed data querying and transformation using SQL
Ensured clean and structured data for Power BI reporting

📊 Key Features
Built 2 interactive dashboards:
Customer Insights Dashboard
Transaction Analysis Dashboard
Developed KPIs:
Revenue, Interest, Transaction Count, Transaction Amount
Implemented dynamic filters (Quarter, Gender, Card Type, Income Group)
Performed customer segmentation based on:
Age, Salary, Job Type, Education, Marital Status
Visualized spending trends and transaction behavior

📈 Impact & Insights
Identified 28.8% Week-over-Week revenue growth
Analyzed 667K+ transactions to uncover patterns
Found male customers contribute ~31M vs 26M (female)
Observed Blue & Silver cards contribute ~93% of transactions
Highlighted TX, NY, CA contributing ~68% revenue
Identified top spending categories: Bills, Entertainment, Fuel
Detected low online transaction usage → digital opportunity
Found age group 30–50 as highest revenue contributors
Evaluated customer acquisition cost across card types

📌 Business Value
Enables customer targeting & segmentation
Helps optimize marketing and acquisition cost
Supports revenue trend analysis
Improves data-driven decision-making
