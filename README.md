# 💳 Credit Card Financial Dashboard 🚀

## 🚀 Overview
Developed an interactive **Credit Card Analytics Dashboard** using **Power BI, SQL, and Excel** to analyze customer behavior, transaction trends, and revenue performance.  

Used Excel data, created structured tables in SQL, and connected it to Power BI for analysis and visualization.

---

## 🛠️ Tech Stack
- **Power BI** – Data visualization & dashboard development  
- **SQL** – Table creation, data loading, and querying  
- **Excel** – Data source and preprocessing  

---

## ⚙️ Data Handling (SQL Work)
- Created **SQL tables** from raw Excel data  
- Loaded and structured data into the database  
- Performed **basic querying and transformations using SQL**  
- Connected SQL data to Power BI for reporting  

---

## 📊 Key Features
- Built **2 interactive dashboards**:
  - Customer Insights Dashboard  
  - Transaction Analysis Dashboard  

- Developed KPIs:
  - Revenue  
  - Interest  
  - Transaction Count  
  - Transaction Amount  

- Implemented dynamic filters:
  - Quarter, Gender, Card Type, Income Group  

- Performed customer segmentation based on:
  - Age, Salary, Job Type, Education, Marital Status  

- Visualized:
  - Spending trends  
  - Transaction behavior  

---

## 📈 Impact & Insights
- Identified **28.8% Week-over-Week revenue growth**  
- Analyzed **667K+ transactions** to uncover patterns  
- Found male customers contribute **~31M vs 26M (female)**  
- Observed **Blue & Silver cards contribute ~93% of transactions**  
- Highlighted **TX, NY, CA contributing ~68% revenue**  
- Identified top spending categories:
  - Bills, Entertainment, Fuel  

- Detected **low online transaction usage → digital opportunity**  
- Found **age group 30–50 as highest revenue contributors**  
- Evaluated **customer acquisition cost across card types**  

---

## 📌 Business Value
- Enables **customer segmentation and targeting**  
- Helps optimize **marketing strategies**  
- Supports **revenue trend analysis**  
- Improves **data-driven decision-making**
